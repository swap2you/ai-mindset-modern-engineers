# YouTube Script Outline Template

This template provides a structure for creating 10-minute long-form YouTube videos.

## Video Structure (10 minutes)

### Hook (0:00 - 0:30)
- **Attention grabber**: Start with a question, surprising fact, or relatable problem
- **Preview**: Briefly mention what viewers will learn
- **Value proposition**: Why this matters to them

### Introduction (0:30 - 1:30)
- **Context setting**: What problem or opportunity are we addressing?
- **Who this is for**: Target audience clarification
- **What we'll cover**: Quick overview of main points

### Main Content (1:30 - 8:30)

#### Point 1 (1:30 - 3:00)
- **Concept explanation**
- **Why it matters**
- **Quick example or demonstration**

#### Point 2 (3:00 - 5:00)
- **Deeper dive into the concept**
- **Common pitfalls or misconceptions**
- **Practical application**

#### Point 3 (5:00 - 7:00)
- **Advanced consideration or edge case**
- **Real-world scenario**
- **Best practices**

#### Point 4 (7:00 - 8:30)
- **Integration with other concepts**
- **Next steps or progression**
- **Recap of key takeaways**

### Call to Action (8:30 - 9:30)
- **Link to GitHub repository**: Specific week or file
- **Encourage engagement**: Comments, questions, likes
- **Preview next week**: Tease upcoming content

### Outro (9:30 - 10:00)
- **Final summary**: One-sentence takeaway
- **Subscribe reminder**
- **Thank you message**

## Content Guidelines

- **Pace**: Speak clearly, allow pauses for reflection
- **Visuals**: Use screen shares, diagrams, or code examples when helpful
- **Examples**: Include concrete, relatable examples
- **Transitions**: Smooth transitions between sections
- **Engagement**: Ask rhetorical questions to maintain attention

## Timestamps

Include timestamps in the YouTube description for easy navigation:
- [0:00] Hook
- [0:30] Introduction
- [1:30] Point 1
- [3:00] Point 2
- [5:00] Point 3
- [7:00] Point 4
- [8:30] Call to Action
- [9:30] Outro

## Notes

- Keep language accessible but not condescending
- Use analogies when explaining complex concepts
- Include "aha moments" - insights that click for viewers
- Balance theory with practical application
