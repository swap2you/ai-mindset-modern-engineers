# LinkedIn Post Template

This template provides a structure for creating LinkedIn posts that support each week's content.

## Post Structure

### Option 1: Short Post (Recommended)

**Hook** (1-2 sentences)
- Attention-grabbing opening
- Relatable problem or surprising insight

**Body** (3-5 sentences)
- Core concept or insight
- Why it matters
- Practical application or example

**Call to Action** (1 sentence)
- Link to GitHub repository
- Specific week or file reference
- Engagement prompt

**Hashtags** (3-5 relevant hashtags)

---

### Option 2: Carousel Post Outline

**Slide 1: Hook**
- Compelling question or statement
- Visual: Diagram or key concept

**Slide 2-4: Main Content**
- One key point per slide
- Visual: Supporting diagram or example
- Keep text minimal

**Slide 5: Call to Action**
- Link to GitHub
- Specific resource reference
- Engagement prompt

**Caption**: Full explanation with context

## Content Guidelines

- **Keep it concise**: LinkedIn users scroll quickly
- **Lead with value**: What will they learn or gain?
- **Use examples**: Concrete examples resonate better
- **Include CTA**: Always link back to GitHub
- **Engage**: Ask questions to encourage comments
- **Professional tone**: Calm, teaching-focused, no hype

## Best Practices

- Post at optimal times (typically Tuesday-Thursday, 8-10 AM)
- Include relevant hashtags (3-5 is ideal)
- Respond to comments promptly
- Cross-reference with YouTube video when available
- Use visuals when possible (diagrams, code snippets)

## GitHub Link Format

Always link to the specific week:
```
https://github.com/[username]/ai-mindset-modern-engineers/tree/main/01-weeks/w##-topic-name
```

Or link to specific files:
```
https://github.com/[username]/ai-mindset-modern-engineers/blob/main/01-weeks/w##-topic-name/02-exercises.md
```
