# YouTube Description Template

This template provides a structure for YouTube video descriptions that support each week's content.

## Description Structure

### Title Placeholder
[Week ##] [Topic] - [Key Takeaway]

### Description

**Hook** (1-2 sentences)
- What viewers will learn
- Why it matters

**Main Content** (2-3 paragraphs)
- Overview of topics covered
- Key takeaways
- Who this is for

**Timestamps** (if applicable)
- [0:00] Introduction
- [X:XX] Topic 1
- [X:XX] Topic 2
- etc.

**Resources**
- GitHub repository link
- Specific week or file links
- Related resources

**Call to Action**
- Subscribe reminder
- Engagement prompts
- Next video preview

**Hashtags** (3-5 relevant hashtags)

## Timestamps Best Practices

- Include timestamps for videos longer than 5 minutes

## Content Guidelines

- **Clear and scannable**: Use formatting (bold, emojis) to break up text
- **Value-focused**: Lead with what viewers will gain
- **Actionable**: Include specific resources and links
- **Engaging**: Encourage comments and discussion
- **Professional**: Maintain teaching-focused tone

## SEO Considerations

- Include relevant keywords naturally
- Use hashtags strategically (3-5 is ideal)
- Include related terms in description
- Link to related content

---

**Remember**: The description should complement the video and drive viewers to the GitHub repository for deeper learning.
